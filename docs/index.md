LandPPT 是一个 AI 驱动的 PPT 制作平台，本说明站覆盖了从需求建模、模板管理到运行配置的全部能力。你可以按“入门 → AI 流程 → 资源管理 → 系统配置”的顺序阅读，也可以直接跳转到需要的章节。

![image](https://img.pub/p/4b07dee6cf3cef023bd8.png)

## 文档结构

- **入门**：了解产品能力与部署方式
- **AI 流程**：需求建模、大纲生成、在线编辑
- **资源管理**：模板与图床的二次封装
- **系统配置**：Env / API / 研究服务等统一设置
